# E-form

为现在的公司所写的快速填写实验记录的工具，但当时很多问题没有考虑，以及当时处于初学者阶段，有些工具不了解，从设计上就欠缺一些东西，所以目前并不实用，暂时作为1.0版本，准备着手重构项目。
前端：vue3 + element plus + NutUI
后端：python(flask，flask-sqlalchemy)
通过这个项目积累了不少经验，也通过实践学会了不少东西，下个项目会更好，敬请期待。
